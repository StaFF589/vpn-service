## 📲 Установка и настройка

### Android

#### Вариант 1: v2rayNG (рекомендуется)

1. **Скачайте приложение:**
   - [GitHub](https://github.com/2dust/v2rayNG/releases) (последняя версия)
   - Или [Google Play](https://play.google.com/store/apps/details?id=com.v2ray.ang)

2. **Добавьте подписку:**
   - Откройте v2rayNG
   - Нажмите `+` (плюс) в правом верхнем углу
   - Выберите **"Подписаться"**
   - Вставьте ссылку из бота
   - Нажмите **OK**

3. **Подключитесь:**
   - Выберите сервер из списка
   - Нажмите круглую кнопку внизу
   - Готово! 🎉

#### Вариант 2: Hiddify

1. Скачайте [Hiddify](https://github.com/hiddify/hiddify-next/releases)
2. Откройте приложение
3. Нажмите **"Добавить из буфера обмена"**
4. Вставьте ссылку
5. Нажмите **Подключиться**

### iOS (iPhone/iPad)

#### Вариант 1: V2Box

1. **Скачайте V2Box:** [App Store](https://apps.apple.com/app/v2box-v2ray-client/id1551169038)

2. **Добавьте конфигурацию:**
   - Откройте V2Box
   - Нажмите **"Add Configuration"**
   - Выберите **"Subscribe"**
   - Вставьте ссылку из бота
   - Нажмите **"Add"**

3. **Подключитесь:**
   - Выберите сервер
   - Переключите тумблер вверху
   - Нажмите **"Connect"**

#### Вариант 2: Shadowrocket (платное)

1. Купите [Shadowrocket](https://apps.apple.com/app/shadowrocket/id932747118) ($2.99)
2. Откройте приложение
3. Нажмите `+` в правом верхнем углу
4. Выберите **"Type: URL"**
5. Вставьте ссылку
6. Нажмите **"Done"**
7. Включите переключатель

#### Вариант 3: Potatso Lite (бесплатное)

1. Скачайте [Potatso Lite](https://apps.apple.com/app/potatso-lite/id1239860606)
2. Добавьте конфигурацию через ссылку
3. Подключитесь

### Windows

#### v2rayN

1. **Скачайте v2rayN:** [GitHub](https://github.com/2dust/v2rayN/releases)
   - Скачайте `v2rayN-With-Core.zip`
   - Распакуйте архив
   - Запустите `v2rayN.exe`

2. **Добавьте подписку:**
   - В меню выберите **"Подписки"** → **"Добавить подписку"**
   - Вставьте ссылку
   - Нажмите **OK**
   - Нажмите **"Обновить подписки"**

3. **Подключитесь:**
   - Выберите сервер
   - Нажмите **"Установить системный прокси"**
   - Готово!
